Coins Deposit Karne Ka Mukammal Guide

Coins Deposit Center mein aapka khushamdeed! Yahan aap Pakistani Rupees (PKR) ke zariye fori taur par coins khareed sakte hain. **Naya Rate:** Ab **6 PKR** ke badle aapko **1 Coin** milega. Yeh rate aapki earning activities ko zyada value deta hai.

Deposit Form aur History dekhne ke liye, barah-e-meherbani Profile Icon (üë§) par click karke Login/Signup karen.

1. Coins Khareedne Ka Tareeqa

Coins khareedne ka amal bahut seedha hai:

  • Package Chunein: Neeche diye gaye packages mein se apni marzi ki miqdar chunein. Yaad rakhen, agar aap 600 PKR deposit karte hain, to aapko 100 Coins milenge.
  • Raqam Transfer Karen: Payment Methods section mein diye gaye EasyPaisa, JazzCash, ya SadaPay accounts par utni hi raqam (PKR) transfer karen jitne coins aap khareedna chahte hain (Coins ki miqdar x 6).
  • Transaction Details: Raqam transfer karne ke baad, aapko **Transaction ID (TID)** ke saath, **jis account se raqam bheji gayi hai uski tafseelat (naam, number, CNIC)** bhi form mein darj karni hogi.

2. Deposit Request Submit Karna

Login karne ke baad, aapko Deposit Form nazar aayega. Is form ko dhyan se fill karen:

  1. Deposit Method: Woh method chunein jiske zariye aapne raqam bheji hai.
  2. Bheji Gayi Raqam: Woh sahi raqam darj karen jo aapne transfer ki hai.
  3. Sender Details: Raqam bhejney walay ka naam, account number, aur CNIC number darj karen.
  4. Transaction ID: Woh TID ya Hash code darj karen jo aapko transfer ke baad mila tha.
  5. Submit: Request submit karen.

3. Verification Aur Coins Ka Milna

Aapki request submit hone ke baad, woh 'Pending' status mein chali jayegi. Humari team fori taur par aapki Transaction ID aur Sender Details ko verify karegi. Verification mukammal hone par:

  • Aapki request ka status 'Approved' ho jayega.
  • Coins (Bheji gayi Raqam / 6) ke hisab se aapke wallet mein add kar diye jayenge.
  • Aap Deposit History section mein apni tamam requests ka status track kar sakte hain.
